The difference between both of these is the semantic role. Deleted text has semantic role. It marks text as deleted, and screen readers reads them as deleted. The del tag fits when you want to mark removed text in a document or revision.

Striked words have no value, meaning there is no semantic role and screen readers skips their meaning. You use the s tag when you want to show words that lost their weight but must still remain.
